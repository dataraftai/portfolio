### Projects

####  [Telecom Customer Churn Prediction & Survival Analysis](https://github.com/dataraftai/Customer_churn_prediction)

In this project, I analyzed how the probability of customer churn changes over time and built ML models to predict churn.  
An XGBoost classifier was trained with class imbalance handling, and the model was deployed as a FastAPI service and Dockerized.

https://github.com/dataraftai/Customer_churn_prediction



#### [Medical Insurance Risk Prediction API](https://github.com/dataraftai/Health-Insurance-Risk-Prediction-API)

Built a production-ready REST API to predict high-risk and low-risk insurance customers using Logistic Regression.  
Implemented adjustable probability thresholds and Dockerized the application with Swagger documentation.
